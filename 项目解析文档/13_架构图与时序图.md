## 真实模式时序图

```mermaid
sequenceDiagram
    participant User as Buyer Message
    participant Entry as main.py
    participant Live as XianyuLive
    participant API as XianyuApis
    participant Ctx as ChatContextManager
    participant Bot as XianyuReplyBot

    Entry->>Live: 初始化真实模式
    Live->>API: 获取 token / 商品信息
    Live->>Ctx: 读取历史消息与商品缓存
    Live->>Bot: 传入用户消息、商品信息、上下文
    Bot-->>Live: 返回回复文本
    Live->>Ctx: 写入本轮对话
    Live-->>User: 发送回复
```
