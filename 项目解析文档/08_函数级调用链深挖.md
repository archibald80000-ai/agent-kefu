# 08 函数级调用链深挖

## 核心调用链

公开版可以按下面这条链去理解代码：

1. `main.py` 读取环境变量并选择运行模式
2. 真实模式下实例化 `XianyuLive`
3. `XianyuLive` 负责：
   - 建连
   - 心跳
   - 收消息
   - 取商品信息
4. 消息进入 `XianyuReplyBot.generate_reply`
5. `XianyuReplyBot` 调用分类 / 价格 / 技术 / 默认角色
6. 回复结果回到 `main.py` 发送，并同步写入上下文

## 为什么这条链重要

学习者只要看懂这条链，就能理解：

- 哪些是平台专属逻辑
- 哪些是大模型专属逻辑
- 哪些是状态管理逻辑

## 重构时最适合切的刀口

- 入口配置处理
- 平台 API 封装
- Agent 分发
- 消息存储
